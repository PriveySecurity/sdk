# Privey

Privey is a privacy-focused decentralized application (dApp) system built on Solana, designed to enhance transaction privacy and ensure strong unlinkability between deposits and withdrawals.

By addressing the transparency limitations of public blockchains, Privey enables users to interact on-chain while significantly reducing traceability and preserving financial confidentiality.

## Overview

Public blockchains expose transaction histories by default, making it possible to trace asset flows across wallets. Privey introduces a privacy-preserving architecture that cryptographically separates deposits from withdrawals, preventing direct on-chain linkage.

Privey is designed for users, developers, and protocols that require privacy, security, and scalability within the Solana ecosystem.

## High-Level Architecture

1. Deposit  
   Users deposit assets into the Privey system and receive cryptographic commitments.

2. Privacy Pool  
   Deposits are pooled to obfuscate individual transaction paths.

3. Withdrawal  
   Users withdraw assets using privacy proofs without revealing deposit origins.

This approach ensures strong unlinkability while remaining fully on-chain.
